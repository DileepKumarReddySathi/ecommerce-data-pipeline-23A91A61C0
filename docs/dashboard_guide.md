# Dashboard User Guide

## Accessing the Dashboard

### Power BI
- File: dashboards/powerbi/ecommerce_analytics.pbix
- Tool: Power BI Desktop (Free)

---

## Dashboard Pages

### Page 1: Executive Overview
Purpose: High-level business metrics

Key Metrics:
- Total Revenue
- Total Transactions
- Average Order Value
- Profit Margin

Visuals:
- Monthly revenue trend
- Top categories
- Payment methods
- Sales map

---

### Page 2: Product Performance
Purpose: Product analysis

Insights:
- Top products generate most revenue
- Electronics has highest margin
- Price vs profit correlation

---

### Page 3: Customer Analytics
Purpose: Customer behavior analysis

Insights:
- High-value customers dominate revenue
- Clear segmentation patterns
- Retention trends

---

### Page 4: Trends & Geography
Purpose: Time & location analysis

Insights:
- Top states drive revenue
- Weekend sales higher
- Seasonal trends visible

---

## Filters
- Date
- Category
- State
- Payment Method

## Refreshing Data
1. Run pipeline
2. Open Power BI
3. Click Refresh
4. Save dashboard

## Business Recommendations
- Focus on top-performing regions
